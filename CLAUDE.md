# Project conventions for Claude Code

## No AI attribution in any artifact

Do **not** add AI-attribution lines to anything that lands in this repo or in any public-facing artifact. This includes:

- Git commit messages — no `Co-Authored-By: Claude ...`, no "Generated with Claude Code", no `🤖` markers.
- Pull request descriptions — no "Generated with Claude Code" footer.
- README.md, docs, code comments, error messages — no mentions of Claude, Anthropic, or AI tooling as author/contributor.
- Issue comments, changelogs, release notes — same.

Commit messages: subject line + (optional) body bullets. Nothing else.

This applies to **every session** working in this repo, not just the one that set the rule.
